Server finden, prüfen und mit klaren Daten verfolgen.

ASA Server Eye bündelt die Suche nach offiziellen Servern, Serverdetails, Favoriten und Player Sightings in einer fokussierten mobilen Anwendung.

Die App in der Anwendung.

Die Übersicht zeigt echte Projektansichten für Anmeldung, Server Discovery, Serverdetails und Einstellungen.

Auf die wichtigsten Abläufe konzentriert.

Die App reduziert die Serverauswahl auf Informationen und Aktionen, die vor und während einer Spielsitzung relevant sind.

Server Discovery

Offizielle Server und Karten durchsuchen und zentrale Angaben direkt vergleichen.

Serverdetails & Favoriten

Karte, Auslastung und offiziellen Status prüfen sowie häufig genutzte Server speichern.

Player Sightings

Sichtungen zu einem Server aufrufen und neue Beobachtungen strukturiert melden.
