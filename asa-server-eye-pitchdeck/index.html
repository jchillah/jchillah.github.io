Problem & Lösung

Problem

ARK-Spieler müssen offizielle Server wiederholt suchen und relevante Informationen vor dem Spielstart vergleichen.

  • Servernamen, Karte und Auslastung sind schwer überschaubar.
  • Wichtige Server sollen ohne erneute Suche erreichbar bleiben.
  • Player Sightings benötigen einen klaren Serverbezug.

Lösung

ASA Server Eye bündelt Server Discovery, Detailansichten, Favoriten und Player Sightings in einer fokussierten Flutter-App.

  • Offizielle Serverübersicht
  • Suche nach Servern und Karten
  • Serverdetails und Favoriten
  • Player Sightings pro Server
