ASA Server Eye

Flutter Companion-App für ARK: Survival Ascended Serverdaten

Jchillah’s Design & Coding Forge · Michael Winkler · 20.07.2026

Problem & Lösung

Problem

ARK-Spieler müssen offizielle Server schnell finden, wiedererkennen und relevante Serverinformationen prüfen. Ohne mobile Übersicht müssen Serverdaten wiederholt gesucht werden.

  • Serverstatus und Details sind unterwegs schwer vergleichbar.
  • Wichtige Server sollen ohne erneute Suche verfügbar bleiben.
  • Favoriten, Watchlists und Alerts brauchen eine stabile App-Grundlage.

Lösung

ASA Server Eye bündelt offizielle ARK: Survival Ascended Serverdaten in einer mobilen Flutter-App mit Suche, Detailansichten und Favoriten.

  • Offizielle Serverübersicht
  • Server-Suche und Filterung
  • Detailansicht pro Server
  • Favoriten für wichtige Server
