ASA Server Eye

Flutter Companion-App für ARK: Survival Ascended Serverdaten

Jchillah’s Design & Coding Forge · Michael Winkler

Problem & Lösung

Problem

ARK-Spieler müssen offizielle Server schnell finden, vergleichen und wiedererkennen.

  • Serverdaten sind unterwegs schwer vergleichbar.
  • Wichtige Server sollen ohne erneute Suche verfügbar sein.
  • Favoriten und Alerts brauchen eine stabile Grundlage.

Lösung

ASA Server Eye bündelt offizielle Serverdaten in einer mobilen Flutter-App.

  • Serverübersicht
  • Suche und Filterung
  • Detailansichten
  • Favoriten
